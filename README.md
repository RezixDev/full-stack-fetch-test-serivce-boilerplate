# Full Stack Fetch Test Service Boilerplate

## Server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run server.ts
```

This project was created using `bun init` in bun v1.1.13. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Frontend

To install dependencies:

```bash
cd frontend
npm install
```

To run:

```bash
npm run dev
```
